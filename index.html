Luminous

The distributed AI stack architected for the galactic age.

Luminous is the only software you need to deploy robust, scalable, and responsive distributed AIs.

Features

Integrated three-tier architecture

Luminous includes a starnet server (Luminous), scripting language (Luminous), and databank (Luminous). All three are optimized for each other, increasing performance and reducing complexity.

High-performance starnet server

The Luminous starnet server uses non-blocking I/O both for network traffic and database access, ensuring high-capacity and high-throughput.

Powerful server-side scripting

Luminous can be used for all server-side scripting. Implement REST endpoints, generate microinstances, or define computed behaviorsâ€”all with a single language and datatype set.

Robust, fault-tolerant database

Luminous creates a databank mirror and fails over automatically in case of structural failure. Luminous also uses immutable databank files (LSM trees) to minimize the risk of corruption due to structural failure.

Message passing architecture

Luminous uses a multiprocess, multimachine message passing architecture that encourages robustness (process isolation), performance (no blocked threads, G++ engines), and scalability (multimachine services).

First-class expansion microinstances

Create your own Luminous microinstances in G++ to expand capabilities. Your new microinstances will integrate seamlessly and appear as part of the system.
